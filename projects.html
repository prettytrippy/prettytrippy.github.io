Medical Projects

SayAhh Signal Processing

I wrote a large amount of signal processing code for Emercent Technologies' SayAhh app, an application that uses sound to measure respiratory and vocal health.

I implemented a variety of speech processing algorithms, such as a Dart version of the YIN algorithm for fundamental frequency estimation. My algorithms were used to gather a number of other speech biomarkers, such as jitter, shimmer, mel-frequency cepstral coefficients, and some other characteristics we deemed useful. I also designed a data pipeline to transport this data from a mobile device to a remote server for processing and storage.

Binaural Beats Entrainment

I designed and ran an exploratory study on the use of binaural beats for brain-wave entrainment.

Entrainment happens when interacting oscillatory systems start to synchronize, as destructive interference cancels out their unlike frequency components, leaving the ones they have in common. Sinces neurons fire in designated frequency bands, it's plausible that low frequency sound waves (like the ones heard in binaural beats) would be able to entrain brain waves into desirable states (relaxed, focused, etc.). If I could show that this happens, we could create an application to entrain brainwaves, to assist people with sleep, focus, learning, and other cognitive tasks.

The system I designed consisted of a wearable rig using the OpenBCI Ganglion board for EEG data collection, a software module to play binaural beats at target frequencies, and a data processing toolkit to look for evidence of entrainment. Although the results were negative, I am grateful for this opportunity to conduct research in an industrial setting.

Calibration Server

I created an internal web server for Emercent Technologies to store and process data from spirometers and other proprietary medical devices. The server stored files in an authenticated MongoDB database. I also implemented a variety of machine learning algorithms, to compare their effectiveness in calibrating spirometric equipment. Algorithms included the Yeh algorithm, a transformer model, and various regression models.
