Computational Medicine

  • Calibration Transformer

    Affiliation: Emercent Technologies

    Spirometers collect information about the amount of air a subject is able to exhale and inhale over time. However, the values they pick up are not quite right, for various physical reasons. To calibrate a spirometer, it is necessary to find its conductance array, a list of co-efficients that will correctly map observed values to real values. This calibration is tricky, with many approaches requiring large amounts of data and computation. During my time at Emercent, I created a transformer model and dataset for calibrating spirometric data. The model architecture was custom made; calculating loss required some extra steps, since we do not have ground truth answers, but only known properties of an answer. I began training the model shortly before returning to school, so it is still a work in progress.

  • BreathMetrics

    Affiliation: Area 10 Labs

    I assisted in the development of a mobile application for spirometer analysis. The app was developed using the Unity framework, and was able to read in time-series respiratory data, use techniques from digital signal processing to glean useful information from the data, and create comprehensible graphics for a user. This data was stored as a user history, from which a user was able to visualize changes in various attributes (like FIV or PEF) over time. I also worked with respiratory scientists at Mayo Clinic to create algorithms for spirometric calibration and visualization. In addition to this application, I created mathematical models of human respiration to test our approach, and assisted with the fabrication and soldering of medical hardware.

  • Graphical Methods for Author Name Disambiguation

    Affiliation: MSU DeepThought Initiative

    Given a large enough sample of academic publications, there are bound to be authors with the same name, or at least the same first initial and last name (as per academic convention). Worse still, a single academic might use a nickname, change their first name, get married, or go by multiple names for some other reason.
    Databases like S2ORC and the NASA ADS contain sparse metadata about the publications that they index, with informative attributes like abstracts, keywords, and affiliations often missing or misattributed. Thus, it is necessary to create an author name disambiguation (AND) system capable of working with few publication attributes.
    The system I've devised trains a random forest classifier to disambiguate authors, using graphical information about the co-author network surrounding a given publication, such as branching factor, path length, and connectivity. Currently, this approach manages extremely high accuracy when validated using labeled ORCID data, despite using only titles and author names.

  • Akashic

    Hobby Project

    Retrival-augmented chatbot that runs on a single laptop. Uses the llama.cpp Python binding + quantized models to reply to user prompts from a web interface. Documents are uploaded to user-defined collections, and the user gets to decide which collections the model sees each prompt.   • Ludwig Wittgenstein's Philosophy of Religion

    Affiliation: Epistemai

    As head staff writer for the UMN Undergraduate Journal of Philosophy, Epistemai, I wrote about Ludwig Wittgenstein's philosophy of religion, as expressed in Remarks on Frazer's 'The Golden Bough'. I mainly focus on defending Wittgenstein against the critiques leveled at him by John Searle, as well as offering an interpretation of his central argument. This piece was published in the sixth volume of the journal, which is free to download here.
